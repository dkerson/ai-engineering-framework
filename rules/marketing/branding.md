# Branding

## Regras

- Respeitar identidade visual existente quando houver.
- Definir tom de voz antes de escrever copy extensa.
- Manter consistencia entre logo, cores, tipografia, iconografia, imagens e linguagem.
- Nao copiar marcas, layouts proprietarios ou assets protegidos.
- Para projetos novos, propor uma identidade moderna sem depender de referencias fechadas.

## Checklist rapido

- [ ] Tom de voz documentado
- [ ] Cores e tipografia coerentes
- [ ] Logo e icones seguem mesmo estilo
- [ ] Linguagem textual consistente
- [ ] Inconsistencias registradas no Working Context
