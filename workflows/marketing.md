# Workflow: Growth & Brand Intelligence

> Branding, copy, marketing, SEO, landing pages e assets. Modo padrao: **Standard**; escalar para **Review** quando houver implementacao frontend/mobile, design system ou auditoria de conversao.

## Trigger

`marketing` / `growth` / `brand` / `landing-page` / `site` / `home-publica` / `copy` / `seo` / `assets` / `product-audit`

## Modo

| Condicao | Modo |
|----------|------|
| Copy, microcopy, SEO ou recomendacao pontual | Fast/Standard |
| Landing page, site, portal ou tela publica | Standard/Review |
| Redesign, rebranding, auditoria ampla ou >3 modulos | Review/Council |

## Pipeline inteligente

```text
Project Style Analyzer -> Knowledge Engine -> Brand Strategist
-> Brand Identity/Voice/Consistency conforme necessidade
-> Content Strategist -> Copywriter -> Headline/CTA/FAQ/Storytelling conforme necessidade
-> UX Writer -> Microcopy Specialist conforme interface
-> Product/UX/UI/Design System conforme disponivel
-> Asset Intelligence -> Asset Curator/Open Source Asset Finder conforme necessidade
-> Landing Page Specialist -> SEO Specialist -> Growth Strategist/Trust Builder/Social Proof
-> Frontend/Mobile* -> Visual Consistency Reviewer -> Brand Reviewer
-> Marketing Reviewer -> Conversion Optimizer -> Product Excellence -> QA
```

\* Frontend/Mobile conforme stack e necessidade real.

## Pipeline de assets

```text
Asset Intelligence -> Logo Manager* -> Icon/Image/Illustration Curator*
-> Image Optimizer -> Brand Reviewer
```

\* Condicional conforme tipo de asset decidido.
