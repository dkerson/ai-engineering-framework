# Workflow: Framework Operating System

> Governance workflow for questions and decisions about the framework itself.

## Trigger

- "Como esta nosso Framework?"
- "Qual nosso Roadmap?"
- "Quais melhorias recomenda?"
- "Quais skills nunca usamos?"
- "Qual nossa saude?"
- "O que implementamos?"
- "O que ainda falta?"

## Pipeline

```text
Framework Operating System
-> Framework Reviewer
-> Anti Pattern Detector
-> Pattern Extractor*
-> Recommendation Engine
-> Framework Optimizer
-> Report
```

`*` Only when enough repeated evidence exists.

## Rules

- Read FOS living documents first.
- Never implement automatically.
- Register and recommend.
- Wait for user approval.
