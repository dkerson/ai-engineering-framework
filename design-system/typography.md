# Tipografia

## Escala (Greenfield)

| Nível | Size | Weight | Uso |
|-------|------|--------|-----|
| display | 32px | 600 | Título de página |
| h1 | 24px | 600 | Seções principais |
| h2 | 20px | 600 | Subseções |
| body | 14–16px | 400 | Texto padrão |
| caption | 12px | 400 | Metadados, hints |
| label | 12–14px | 500 | Form labels |

## Regras

- Line-height: 1.4–1.6 corpo; 1.2 títulos
- Máx. 2 famílias (UI + mono para código)
- Truncar com ellipsis em listas densas
- LEGACY: preservar fontes do sistema
