# Inputs

- Label visível (não só placeholder)
- Placeholder = exemplo, não label
- Erro: borda + mensagem abaixo
- Helper text abaixo do campo
- Grupos relacionados com fieldset/legend quando necessário

## Tipos

text · number · date · select · combobox · textarea · checkbox · radio · switch

## A11y

`id` + `for`, `aria-invalid`, `aria-describedby`
