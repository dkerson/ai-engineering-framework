# Botões

| Variante | Uso |
|----------|-----|
| Primary | 1 ação principal por contexto |
| Secondary | Ação alternativa |
| Ghost/Tertiary | Ações frequentes, baixo peso |
| Danger | Exclusão, irreversível |
| Icon | Toolbar, densidade alta |

## Estados

default · hover · focus-visible · disabled · loading

## Regras

- Altura mín: 36px desktop, 44px touch
- Loading: spinner + label ou só spinner com aria-busy
- LEGACY: não trocar componente de botão
