# Checklist: Landing Page

- [ ] Hero comunica oferta, publico e proximo passo
- [ ] Beneficios sao especificos e orientados a resultado
- [ ] Funcionalidades sustentam beneficios
- [ ] Prova social real ou parametrizada
- [ ] CTA primario consistente
- [ ] FAQ cobre objecoes relevantes
- [ ] Contato/formulario sem friccao desnecessaria
- [ ] SEO basico definido
- [ ] Assets seguem identidade visual
