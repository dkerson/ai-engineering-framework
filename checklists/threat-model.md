# Checklist: Threat Model

- [ ] Assets protegidos listados
- [ ] Atores e service accounts listados
- [ ] Trust boundaries identificados
- [ ] Entradas externas mapeadas
- [ ] Abuse cases priorizados
- [ ] Privilege escalation avaliado
- [ ] Cross-tenant leakage avaliado
- [ ] Replay/enumeration avaliados
- [ ] Mitigacoes e risco residual definidos
