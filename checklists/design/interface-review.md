# Checklist — Revisão de Interface

- [ ] Parece software profissional/moderno (para o modo escolhido)
- [ ] Hierarquia visual clara
- [ ] Sem excesso de informação ou cores
- [ ] Legibilidade adequada
- [ ] Componentes consistentes
- [ ] Responsividade desktop/tablet/mobile
- [ ] Transmite confiança
- [ ] Mínimo de cliques para tarefa principal
- [ ] Reuso de componentes maximizado

Skill: `design-reviewer`
