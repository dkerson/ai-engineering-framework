# Checklist: Security Architecture

- [ ] Trust boundaries definidos
- [ ] Dados sensiveis identificados
- [ ] Autenticacao e autorizacao separadas
- [ ] Tenant isolation validado
- [ ] Acoes privilegiadas listadas
- [ ] Fonte autoritativa de permissao definida
- [ ] Logs/auditoria sem secrets ou PII desnecessaria
- [ ] Threat model resumido no Working Context
- [ ] Rollback ou mitigacao emergencial definido
