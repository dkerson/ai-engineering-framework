# Checklist: Mission Recognition

- [ ] Objetivo principal identificado
- [ ] Objetivos secundarios identificados
- [ ] Mission type definido
- [ ] Mission mode definido
- [ ] Dominios envolvidos mapeados
- [ ] Urgencia, impacto, complexidade e risco estimados
- [ ] Mission score preenchido
- [ ] Knowledge Hub sugerido
- [ ] Perguntas evitadas quando objetivo esta claro
- [ ] Mission Brief pronto para o Master Orchestrator
