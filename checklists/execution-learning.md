# Execution Learning Checklist

- [ ] Mission had a reusable lesson, not just a one-off fact.
- [ ] Evidence is short and sanitized.
- [ ] Root cause or corrected assumption is explicit.
- [ ] Destination chosen: learning, pattern, anti-pattern, recommendation, roadmap or metric.
- [ ] Promotion status is recorded.
- [ ] No secrets, full prompts, private payloads or long code blocks were stored.
- [ ] Framework change was implemented only after explicit user approval.
- [ ] Final response mentions any framework learning that affects future work.
