# Checklist — Performance de Query

- [ ] Plano de execução analisado
- [ ] Índices adequados nas colunas de filtro/join
- [ ] Evitar funções em colunas indexadas no WHERE
- [ ] Agregação antes de join quando possível
- [ ] Sem SELECT desnecessário em subqueries
- [ ] Baseline e after medidos
- [ ] Equivalência de resultado validada
