# Capability Health Checklist (COS)

## Registry

- [ ] CAPABILITY_REGISTRY atualizado
- [ ] CAPABILITY_STATUS sincronizado
- [ ] CAPABILITY_ROADMAP reflete prioridades
- [ ] CAPABILITY_REPORT responde perguntas naturais

## Por capability

- [ ] Categoria definida
- [ ] Lifecycle status coerente
- [ ] Health score documentado (14 dimensões)
- [ ] Documentação mínima presente
- [ ] Plugins (se houver) com capability pai

## COS

- [ ] capability-manager nunca auto-implementa
- [ ] FOS atualizado em transições
- [ ] PROJECT_CAPABILITIES nos projetos
