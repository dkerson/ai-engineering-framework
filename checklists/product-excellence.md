# Checklist: Product Excellence

- [ ] UX clara e orientada ao objetivo do usuario
- [ ] UI consistente com o projeto
- [ ] Mobile adequado
- [ ] Desktop produtivo
- [ ] Performance considerada
- [ ] Conversao ou proximo passo claro quando aplicavel
- [ ] Linguagem humana e objetiva
- [ ] Assets consistentes e licenciados
- [ ] Design System respeitado ou lacuna documentada
- [ ] Categorias abaixo de 8 possuem melhoria recomendada
