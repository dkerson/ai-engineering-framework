# Recommendation Engine

## Objetivo

Gerar recomendacoes justificadas para evoluir o framework.

## Categorias

- Nova Skill
- Novo Workflow
- Novo Template
- Nova Rule
- Nova Checklist
- Nova Documentacao
- Nova Mission
- Novo Dominio
- Novo Pattern

## Regras

- Sempre justificar.
- Sempre registrar.
- Nunca implementar automaticamente.
- Aguardar aprovacao do usuario.
