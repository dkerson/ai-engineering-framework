# Anti Pattern Detector

## Objetivo

Registrar riscos de qualidade e governanca do framework.

## Detectar

- Duplicacao
- Fluxos ruins
- Skills redundantes
- Regras conflitantes
- Documentacao repetida
- Desperdicio de tokens
- Arquitetura excessivamente complexa

## Regras

- Registrar sempre.
- Nunca alterar automaticamente.
