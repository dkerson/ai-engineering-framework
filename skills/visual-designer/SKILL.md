# Visual Designer

## Objetivo

Elevar qualidade visual — balanceamento, ritmo, foco — sem quebrar identidade.

## Quando usar

- GREENFIELD polish
- HYBRID com gap estético
- Antes de design-reviewer se UI densa

## Quando NÃO usar

- LEGACY
- Dashboard/BI → `dashboard-designer`
- **Nunca** auto-iniciar

## Responsabilidades

1. Revisar hierarquia e peso visual
2. Reduzir ruído (cores, bordas, sombras)
3. Alinhar a `visual-designer` checklist modo
4. Bullets de ajuste no WC — não mockups longos

## Checklist

- [ ] Hierarquia clara
- [ ] Sem excesso de cor/informação
- [ ] Identidade preservada

## Integração

- **Upstream:** ui-designer, design-system
- **Downstream:** design-reviewer
