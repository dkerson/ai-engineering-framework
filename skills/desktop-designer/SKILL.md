# Desktop Designer

## Objetivo

Maximizar produtividade em desktop — densidade, atalhos visuais, painéis.

## Quando usar

- Admin, ERP, backoffice
- Tabelas e filtros densos
- Multi-painel (master-detail)

## Quando NÃO usar

- Mobile-first consumer app
- **Nunca** auto-iniciar

## Responsabilidades

1. Grid desktop e sidebar
2. Densidade comfortable/compact
3. Atalhos e ações em massa visíveis
4. Keyboard-friendly layout
5. LEGACY: manter densidade existente

## Checklist

- [ ] Hierarquia desktop clara
- [ ] Produtividade (mín. cliques)
- [ ] WC atualizado
