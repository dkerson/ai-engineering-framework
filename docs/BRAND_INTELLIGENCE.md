# Brand Intelligence

> Capacidade do framework para manter identidade, voz e consistencia de marca.

## Objetivo

Garantir que produto, interface, marketing e assets expressem a mesma marca.

## Design DNA expandido

Todo projeto deve considerar:

- Brand DNA
- Communication DNA
- Marketing DNA
- Visual DNA
- Product DNA

## Skills principais

- `brand-strategist`
- `brand-identity-manager`
- `brand-voice-manager`
- `brand-consistency-manager`
- `brand-reviewer`
- `visual-consistency-reviewer`

## Regras

- Respeitar identidade existente.
- Para projetos novos, propor uma identidade original e moderna.
- Nunca copiar marcas, layouts proprietarios ou estilos fechados.
- Registrar lacunas no Working Context.
