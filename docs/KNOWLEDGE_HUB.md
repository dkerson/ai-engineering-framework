# Knowledge Hub

> Biblioteca interna de boas praticas resumidas do AI Engineering Framework.

## Objetivo

Oferecer conhecimento estruturado para as skills consultarem antes de planejar, escrever ou implementar.

## Estrutura

`knowledge/<area>/README.md`

Areas previstas:

- design
- ux
- ui
- product
- architecture
- frontend
- backend
- database
- sql
- powerbi
- flutter
- react
- marketing
- branding
- copywriting
- seo
- performance
- security
- qa
- accessibility
- design-system
- mobile
- desktop
- analytics
- dashboard

## Regras

- O Knowledge Hub nao copia documentacao externa.
- Cada entrada resume principios, heuristicas e checklists internos.
- O Orchestrator usa `knowledge-engine` para consultar somente areas relevantes.
- O resumo consultado deve ir para o Working Context.
