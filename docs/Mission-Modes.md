# Mission Modes

> Modos estrategicos escolhidos pelo SIL antes do Master Orchestrator escolher Fast/Standard/Review/Council.

| Mission Mode | Uso |
|--------------|-----|
| Analysis Mode | Entender produto, problema ou oportunidade |
| Planning Mode | Criar plano, roadmap ou estrategia |
| Implementation Mode | Preparar execucao concreta pelo Orchestrator |
| Validation Mode | Verificar qualidade, risco e resultado |
| Continuous Improvement Mode | Criar backlog evolutivo apos entrega |
| Innovation Mode | Explorar diferencial, proposta ou experiencia nova |
| Emergency Mode | Incidente, urgencia ou alto impacto |
| Learning Mode | Usuario quer entender, documentar ou comparar |

## Relacao com modos do Orchestrator

Mission Modes nao substituem Fast, Standard, Review ou Technical Council.

O SIL escolhe o modo estrategico. O Master Orchestrator escolhe o modo operacional.

Exemplo:

```text
Modernization Mission
Mission Mode: Analysis + Planning + Implementation
Orchestrator Mode: Review
```
