# Rules — ClickUp Plugin

## Regras especificas

1. Seguir `plugins/PLUGIN_POLICY.md` global
2. Nunca armazenar secrets — usar env vars de `MCP.md`
3. Access mode: **read-write** — respeitar limites
4. Risco **medium** — acionar `risk-reviewer` quando write em producao
5. Carregar somente arquivos deste plugin necessarios a missao
